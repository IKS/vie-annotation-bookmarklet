vie-annotation-bookmarklet by szabyg

VIE-based, stanbol-connected enhancer bookmarklet based on annotate.js

Install

Stanbol uri:
Bookmarklet label:
Default CSS Selector:
Configurable CSS selector:

- Go to your web browser (Chrome and Firefox are tested so far) - Make sure your bookmark toolbar is visible - Go to http://dev.iks-project.eu/demos/snml-orf-bookmarklet/ - Customize Stanbol url and label e.g. url: `http://dev.iks-project.eu/stanbolfull` label: `Stanbol Full` would use the stable launcher on our server. - Click 'Generate Bookmarklet' - Drag-and-drop the created link on your Browser Link-toolbar
